# Security Policy

## Reporting A Vulnerability

If you believe you have found a security vulnerability in PromptQuery, please do not open a public GitHub issue with sensitive details.

Instead, email the PromptQuery team at:

```text
security@promptquery.ai
```

Please include:

- A clear description of the vulnerability
- Steps to reproduce the issue
- Affected PromptQuery version
- Operating system
- Database type, if relevant
- Any logs, screenshots, or proof-of-concept details that are safe to share privately

We will review the report and respond as soon as possible.

## Public Issues

For non-sensitive bugs, compatibility issues, and feature requests, please use GitHub Issues.
